# APIExplorer

A minimal frontend project to explore and display data from public APIs.

<!-- ## Live Demo
[api-explorer.vercel.app](https://api-explorer.vercel.app) -->

## Features
- Fetch data from multiple public APIs
- Search and filter results in real time
- Clean card-based display with modal view
- Skeleton loaders and error handling
- Like / save individual cards

## APIs Used
- [JokeAPI](https://v2.jokeapi.dev/) — Random jokes with category filter
- [Cat Facts](https://catfact.ninja/) — Random cat facts

## Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)
